# Chatbot

## Para iniciar el proyecto

Antes de iniciar con el proyecto, ten el Node.js instalado version (v22.11.0) `esta version puede ir cambiando!!!`
Instalar react dentro del proyecto (Si aun no lo tienes!)

# Realizar lo siguiente:

### `npm start`

Ejecuta el comando npm start en la consola con la direccion de donde estes guardando el clone del proyecto

### Dependencias

Es muy probable que te pida instalar librerias, en la consola te aparecera lo que tengas que instalar
